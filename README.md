# Creator OS Landing

Live at https://creator-os.vercel.app

Env vars:
- CHECKOUT_TRIPWIRE_URL / CHECKOUT_FLAGSHIP_URL / CHECKOUT_DFY_URL - buy buttons. Default to Stripe test. Swap to Gumroad URLs when products exist.
- KLAVIYO_PRIVATE_KEY - private API key from Klaviyo account
- KLAVIYO_CREATOR_OS_LIST_ID - waitlist Klaviyo list id (U8Kia6)
- KLAVIYO_POSTBUY_TRIPWIRE_LIST_ID - YkC7MF
- KLAVIYO_POSTBUY_FLAGSHIP_LIST_ID - SkjqDB
- KLAVIYO_POSTBUY_DFY_LIST_ID - Wycehc
- GUMROAD_TRIPWIRE_ID / GUMROAD_FLAGSHIP_ID / GUMROAD_DFY_ID - match Gumroad product permalinks
- VSL_EMBED_URL - iframe src for hero video (optional)
- LAUNCH_UTC_ISO - countdown target (optional)
